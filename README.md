# Joseph Lukasik — Portfolio

A rebuild of [joelukasik.com](https://www.joelukasik.com/) for Joseph "Joe" Lukasik —
composer, multi-instrumentalist, and media artist based in Boulder, Colorado.

**Direction:** *Cinematic* — dark-immersive. Fraunces display serif + Manrope, near-black
`#0a0a0b` with warm gold, ambient glow, scroll reveals, and a spotlight-cursor hero.

Pure static HTML/CSS/JS, no build step. Copy lives in [`assets/content.js`](assets/content.js)
and is injected at load; edit it there to change the page.

## Files
- `index.html` — the site.
- `assets/content.js` — all copy (single source of truth).

## Local preview
Any static server, e.g. `npx serve` from this folder.

> Two earlier alternate directions (Editorial, Atelier) and the internal review switcher
> have been removed now that Cinematic is chosen. They remain in git history if needed.
